# AEM AssetsとGenStudio for Performance Marketingを使用したAdobe GenStudio

>[!VIDEO](https://video.tv.adobe.com/v/3439267/?captions=jpn&learn=on)

Adobe GenStudioとAEM Assets as a Cloud Serviceの強力な機能を活用して、パフォーマンスマーケティングワークフローを変革しましょう。 シームレスな統合により、一元化されたリポジトリからブランド承認済みのアセットにアクセスし、あらゆるキャンペーンやチャネル向けに一貫性のある高品質なコンテンツを作成できます。 これらのツールが、どのようにAdobe supply chainのコンテンツを合理化し、マーケティング効果を高めるかをご確認ください。

## 前提条件

AEM AssetsとGenStudio for Performance Marketingを連携するには、次の点を基礎的に理解する必要があります。

* [Adobe GenStudio for Performance Marketing](https://experienceleague.adobe.com/ja/browse/genstudio-for-performance-marketing)
* [AEM Assets as a Cloud Service](https://experienceleague.adobe.com/ja/docs/experience-manager-cloud-service/content/assets/overview)

設定とワークフローに入る前に、AEM AssetsとGenStudio for Performance Marketingに必要なアクセス権があることを確認してください。

## 基本を学ぶ

AEM AssetsとGenStudio for Performance Marketingの基本を理解しているユーザー向けに設計された、2つの包括的なウォークスルーをご覧ください。 最初のチュートリアルでは、Adobe GenStudioとAEM Assets as a Cloud Serviceをシームレスに統合するために必要なすべてをカバーし、設定プロセスについて説明します。 2つ目は、主要なワークフローについて詳しく説明し、統合を活用してパフォーマンスマーケティングコンテンツを効果的に作成、管理、配信するための主な取り組みを紹介します。
