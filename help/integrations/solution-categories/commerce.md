# [!DNL Commerce]

柔軟な AI を利用したコマースソリューションで、単一のプラットフォーム上にマルチチャネルのコマースエクスペリエンスを構築します。 デジタルストアフロントを構築し、市場リーチを拡大し、顧客の利便性を高め、販売の伸びを促します。顧客がいつでもどこでも、製品やサービスを探索、参照、購入できる、シームレスなオンラインショッピングエクスペリエンスを提供します。
