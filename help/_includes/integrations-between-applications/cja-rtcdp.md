# カスタマージャーニー [!DNL Analytics] と Real-time Customer Data [!DNL Platform] （Real-Time CDP）の統合

{{customer-journey-analytics-description}}

{{real-time-cdp-description}}

Adobeカスタマージャーニー [!DNL Analytics] とAdobe Real-Time CDPを統合する主なメリットには、次のものがあります\。

+ **360 度の顧客表示**：統合されたカスタマージャーニービューのデータを組み合わせ、インタラクションや環境設定を理解します。
+ **リアルタイムパーソナライゼーション**:Adobe Real-Time CDPで関連性の高いエクスペリエンスを即座に提供します。
+ **より深い顧客インサイト**：行動とジャーニーを分析し、データ駆動型の決定を通じて戦略を絞り込みます。
+ **シームレスなデータのアクティベーション**：データのアクティベーションを効率化し、チャネルをまたいでパーソナライズされたエクスペリエンスを可能にします。
+ **マーケティング属性の強化**：タッチポイントの影響を正確に属性し、ROI の測定と最適化を向上させます。

全体的に、Adobeカスタマージャーニー [!DNL Analytics] とAdobe Real-Time CDPの統合により、企業は優れたカスタマーエクスペリエンスを提供し、カスタマーエンゲージメントを促進し、包括的でリアルタイムの顧客インサイトに基づいて情報に基づいたマーケティング上の意思決定を行うことができます。
